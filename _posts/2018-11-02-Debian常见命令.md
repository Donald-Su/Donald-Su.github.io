# Debian下软件安装/卸载

安装软件包：
apt-get install git

卸载软件包：
apt-get remove git

## Oh My Zsh安装/卸载
1. 安装
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

2. 卸载 Oh My Zsh
    uninstall_oh_my_zsh
